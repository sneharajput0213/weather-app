# Weather App 🌦️

A simple and responsive weather app built using HTML, CSS, and JavaScript.  
It uses the OpenWeatherMap API to fetch real-time weather data based on city input.

## 🚀 Features
- Live weather data
- Dynamic weather icons/background
- Temperature info
- Clean UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
